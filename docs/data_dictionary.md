# Data Dictionary

This document defines every column present in the final analytical dataset: `tableau_ready.csv`.

| Column Name | Data Type | Example Value | Business Meaning | Formula / Source |
|---|---|---|---|---|
| **InvoiceNo** | String | '536365' or 'C536379' | A 6-digit integral number uniquely assigned to each transaction. If this code starts with letter 'C', it indicates a cancellation. | Original column |
| **StockCode** | String | '85123A' | A 5-digit integral number uniquely assigned to each distinct product. | Original column |
| **Description** | String | 'WHITE HANGING HEART T-LIGHT HOLDER' | The name/description of the product. | Original column |
| **Quantity** | Integer | 6 | The quantities of each product (item) per transaction. | Original column |
| **InvoiceDate** | Datetime | '2010-12-01 08:26:00' | The day and time when each transaction was generated. | Original column (converted to datetime) |
| **UnitPrice** | Float | 2.55 | Product price per unit in sterling (£). | Original column |
| **CustomerID** | String | '17850' | A 5-digit integral number uniquely assigned to each customer. | Original column (converted from float/int to string) |
| **Country** | String | 'United Kingdom' | The name of the country where each customer resides. | Original column |
| **Revenue** | Float | 15.30 | The total amount of money generated by the specific line item. | `Quantity * UnitPrice` |
| **Year** | Integer | 2010 | The year the transaction took place. | Extracted from `InvoiceDate` |
| **Month** | Integer | 12 | The month (1-12) the transaction took place. | Extracted from `InvoiceDate` |
| **Quarter** | Integer | 4 | The quarter (1-4) the transaction took place. | Extracted from `InvoiceDate` |
| **DayOfWeek** | String | 'Wednesday' | The day of the week the transaction took place. | Extracted from `InvoiceDate` |
| **Hour** | Integer | 8 | The hour of the day (0-23) the transaction took place. | Extracted from `InvoiceDate` |
| **is_uk** | Boolean | True | A flag indicating whether the transaction originated in the United Kingdom. | `Country == 'United Kingdom'` |
| **is_cancelled** | Boolean | False | A flag indicating whether the transaction was a cancellation. | `InvoiceNo` starts with 'C' |
| **Customer_LTV** | Float | 1500.20 | The total lifetime revenue generated by the customer associated with the transaction. | Sum of `Revenue` grouped by `CustomerID` |
| **Customer_Cancellation_Count** | Integer | 2 | The total number of cancellations made by the customer. | Count of cancelled `InvoiceNo` grouped by `CustomerID` |
